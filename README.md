# DenoLib

## Usage

For example:
```sh
deno -A https://denolib.com/denoland/deno_std@v0.2.8/http/file_server.ts
```

```ts
import { test } from "https://denolib.com/denoland/deno_std/testing/mod.ts";
```

## Badge

![DenoLib](https://denolib.com/badge?scope=denolib&repo=denolib.com)
